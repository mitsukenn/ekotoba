# あらかき けんしょう「えことば」ウェブサイト

沖縄を拠点に「えことば」（短い手書きの言葉と花の絵）を制作する、**あらかき けんしょう** の公式サイトです。

やさしい言葉と花の絵の作品アーカイブ、絵本紹介、個展・イベント情報を掲載しています。

## 公開URL

GitHub Pages で公開しています 👉 （公開後にURLをここへ記載）

## 構成

純粋な静的サイトです。ビルドツールやフレームワークは使っていません。

```
.
├── index.html      … ページ本体
├── styles.css      … デザイン
├── script.js       … ギャラリー生成・ライトボックス・スクロール演出
├── .nojekyll       … GitHub Pages 用（Jekyll 処理を無効化）
└── images/         … 作品画像
```

## ローカルで確認する

`index.html` をブラウザで開くだけで動きます。

## 更新のしかた

1. 新しいえことば画像を `images/` に追加する
2. `script.js` の `works` 配列にファイル名とことばを追記する
3. 変更をコミットして GitHub に push すると、自動でサイトに反映されます
